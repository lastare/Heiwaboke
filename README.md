## ðŸš€ Cara Menjalankan Proyek

### Prasyarat

- Android Studio (Versi terbaru disarankan, misal: otter)
- JDK 17 atau 21 (Dikonfigurasi melalui JetBrains Runtime)
- Perangkat Android (Fisik atau Emulator) dengan API Level minimal 27.

### Langkah-langkah

1. **Clone Repository**
   ```bash
   git clone https://github.com/lastare/heiwaboke.git
   ```
2. **Buka di Android Studio**
   Pilih `File > Open` dan arahkan ke folder proyek.
3. **Sinkronisasi Gradle**
   Tunggu hingga proses sinkronisasi selesai. Pastikan koneksi internet stabil.
4. **Run Application**
   Klik tombol hijau **'Run'** di toolbar atas.
